# Narration Guidelines

- Write natural spoken Simplified Chinese unless the user requests another language.
- Keep sentences short and tied to visible beats.
- Describe the current visual state before the next transformation.
- Do not read long on-screen text verbatim.
- Do not introduce symbols before they appear on screen.
- Prefer explanation like "现在我们看到..." or "接下来把..." over bullet-list phrasing.
