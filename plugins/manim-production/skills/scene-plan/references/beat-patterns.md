# Beat Patterns

## Concept explainer

- Beat 1: introduce the object or question
- Beat 2: show the core relation
- Beat 3: vary one parameter or example
- Beat 4: takeaway

## Proof walkthrough

- Beat 1: givens
- Beat 2: first transformation
- Beat 3: key inference
- Beat 4: conclusion

## Function visualization

- Beat 1: axes and base function
- Beat 2: key points or intercepts
- Beat 3: slope or parameter meaning
- Beat 4: summary

## Geometry construction

- Beat 1: base figure
- Beat 2: labels and marks
- Beat 3: relation or theorem
- Beat 4: visual conclusion
