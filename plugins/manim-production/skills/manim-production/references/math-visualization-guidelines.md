# Math Visualization Guidelines

- Keep one focal formula or one focal graphic per beat.
- Use color semantically and consistently. Do not recolor the same variable without reason.
- Prefer highlighting, isolation, or transform matching over adding more text.
- In graph scenes, make axes, labels, and curve colors easy to distinguish.
- In proof scenes, separate givens, intermediate steps, and final conclusion spatially.
- Keep text blocks short. If a sentence is long, move it to narration instead of the canvas.
